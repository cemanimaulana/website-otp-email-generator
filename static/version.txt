REQUIREMENTS:
=============
- Hugo 0.80.0 or higher
- Modern web browser with JavaScript enabled
- Micro.blog account (for deployment)

INSTALLATION:
=============
1. Download all theme files
2. Upload to micro.blog themes section
3. Activate theme in blog settings
4. Access /otp page to use generator

CUSTOMIZATION:
==============
- Edit static/js/otp.js to change supported email
- Modify static/css/main.css for styling changes
- Update i18n/*.json files for translations
- Customize layouts/*.html for structure changes
